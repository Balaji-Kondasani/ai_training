# Linear Algebra for Machine Learning -- NumPy Implementation

A hands-on, code-first guide to every linear algebra concept essential for ML.

## Structure

| Notebook | Topics |
|----------|--------|
| `01_vectors.py` | Vector basics, operations, norms, dot product, cosine similarity |
| `02_matrices.py` | Matrix creation, arithmetic, multiplication, broadcasting |
| `03_matrix_operations.py` | Inverse, determinant, trace, rank, solving linear systems, Normal Equation |
| `04_eigendecomposition.py` | Eigenvalues/eigenvectors, PCA from scratch |
| `05_svd.py` | SVD, image compression, LSA for NLP |
| `06_practical_patterns.py` | Distance matrices, softmax, gradient descent |
| `07_exercises.py` | 7 practice exercises combining all concepts |

## Setup

```bash
pip install -r requirements.txt
```

## How to Use

Run each file sequentially. Every file is self-contained and prints its output
with clear section headers. Read the comments to understand each step.
