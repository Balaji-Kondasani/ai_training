# Probability & Statistics for Machine Learning -- NumPy/SciPy Implementation

A hands-on, code-first guide to every probability and statistics concept essential for ML.

## Structure

| File | Topics |
|------|--------|
| `01_probability_basics.py` | Axioms, counting, conditional probability, Bayes' theorem, Naive Bayes |
| `02_distributions.py` | Discrete & continuous distributions (Bernoulli, Binomial, Poisson, Gaussian, etc.) |
| `03_expectation_variance.py` | Expectation, variance, covariance, correlation, Law of Large Numbers, CLT |
| `04_mle_and_map.py` | Maximum Likelihood Estimation, MAP estimation, fitting distributions |
| `05_information_theory.py` | Entropy, cross-entropy, KL divergence, mutual information |
| `06_sampling_methods.py` | Monte Carlo, bootstrap, permutation tests, MCMC intro |
| `07_exercises.py` | 7 exercises combining all concepts into ML applications |

## Setup

```bash
pip install numpy scipy matplotlib scikit-learn
```

## How to Use

Run each file sequentially. Every file is self-contained and prints output
with clear section headers. Read the comments to understand each step.
